==Licensing==

===Overview===
Click here for a full list of licenses available in Washington state; the most commonly searched licenses have been included in the chart below. If your subject operates in an industry/profession that requires licensing and the license is not covered below use the provided list of licenses in Washington state OR type in *profession* + *license* + *jurisdiction* in Google to identify the license lookup. Reach out to the regional lead in order for this resource to be added to our repository.

===License Sources===

====General License Search====
; Source: Washington State Department of Licensing
; URL: https://professions.dol.wa.gov/s/license-lookup
; Guidance:
:• Includes real estate broker and physician licenses.
:• Search by name or license number for individuals or businesses.

====Bar Membership====
; Source: Washington State Bar Association
; URL: https://www.mywsba.org/PersonifyEbusiness/Default.aspx?TabID=1536
; Guidance:
:• Search by name or license number.
:• Search for disciplinary actions here.

====Attorney Search====
; Source: Washington Courts
; URL: https://dw.courts.wa.gov/index.cfm?fa=home.atty&terms=accept&flashform=0
; Guidance:
:• Search by Bar number.

====CPA - Individuals====
; Source: Washington State Board of Accountancy
; URL: https://data.wa.gov/stories/s/CPA-Search-Board-of-Accountancy/pqu3-uhwj
; Guidance:
:• Search by name or by license number.

====CPA - Firms====
; Source: Washington State Board of Accountancy
; URL: https://data.wa.gov/stories/s/CPA-Firm-Search-Board-of-Accountancy/gzrx-79x2
; Guidance:
:• Search by name or by license number.

====Healthcare Providers====
; Source: Washington State Department of Health
; URL: https://fortress.wa.gov/doh/providercredentialsearch/
; Guidance:
:• Includes physicians, registered nurses, and pharmacists.
:• Search by name or credential number.

====Insurance====
; Source: Washington State Office of the Insurance Commissioner
; URL: https://fortress.wa.gov/oic/consumertoolkit/search.aspx?searchtype=aac
; Guidance:
:• Search for an agent, agency, or company by name or license number.

====Contractor, Trade Person or Business====
; Source: Washington State Department of Labor and Industries
; URL: https://secure.lni.wa.gov/verify/
; Guidance:
:• Search by name from the drop-down menu (or whichever search criteria is fitting).
